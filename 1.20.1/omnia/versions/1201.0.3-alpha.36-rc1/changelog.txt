**This is a release candidate update!**

While this is technically still a minor update in my book, there are breaking changes due to mod removal and thus it is not backwards compatible.

For example, Fruitful Fun has been removed as it has been causing crashes in a server environment.

Additional mods have also been added.

### Fixed

- 2026.03.05
  - Fixed Color Yellow's hex code mechanical crafting recipe being incorrect. I'm afraid to know how long it has had the incorrect recipe for now and how it probably made it impossible to craft.
- 2026.02.xx
  - Fixed not being able to craft a normal Minecraft Chest from modded planks. The recipe *should* exist by default, but it seems to be getting removed by an external mod and I've already taken the time to check if any of my scripts do remove the vanilla Chest recipe... but none of them do.
  - Fixed the Witch Eye for End Remastered being uncraftable if Extended Crafting is not present.
  - Fixed conflict between Carry On and Easy Villagers blocks by adding the blocks to the blacklist config.
  - Finally fixed Lightman's Currency villager trades.
  - Partially migrated tooltips from Create's tooltip registry into KubeJS tooltip generation. [This took the bulk of this update's time.]

### Added

- 2026.02.xx - 2026.03.xx
  - Added Egg duplication recipe to reduce the reliance on Chicken entity farms later on when automating Blaze Cake Bases.
  - New methods for making Blaze Cake Base when compacting together in a Basin:
    - Potato, Sugar, Cinder Flour
    - Rice, Sugar, Cinder Flour
  - Added specific block duplication recipe for Biomes o' Plenty such as:
    - Black Sand
    - Orange Sand
    - White Sand
    - etc.
  - Added duplication recipes for all of Alex's Caves cave tablets.
  - Added a recipe to obtain Cardiac's Life Bottles.
  - Added 2 hidden Create blocks and their textures: Refined Radiance Block and Dark Steel Block.
  - Added Minecraft Classic Wool palette with carpets too.

### Changed

- 2026.02.xx - 2026.03.xx
  - Liquid Null from Biomes o' Plenty can now be bottomless.
  - Null Matter from crushing Crushing Wheels is now guaranteed to yield 1x Null Matter instead of being a 12% chance result.
  - Decreased Lunar event occurence chance from 40% to 25%.
  - Increased weight of Blood Moon events from 10 to 30.
  - Decreased weight of White Moon events from 50 to 45.
  - Hopefully disabled Big Moon events by setting weight from 25 to 0.
  - Players can no longer sleep during Crimson Moons.
  - Increased weight of Crimson Moons from 5 to 7.
  - Wall Lanterns from Amendments have been added to Create's Wrench Pickup tag.
  - Changed default keybind for GuideMe's `Open Guide for Items` action from `[W]` to `[G]`.
  - Lodestones can no longer be made with Netherite Ingots.
  - Added the `#minecraft:leaves` tag to the `#kubejs:compost_eligible` item tag so that leaves can be used to make Dirt.
  - Retextured Tenebris Punctura.
  - Create's Zinc Ore now has emissive texture support with Fusion.
  - Changed how projectiles are spawned from certain weapons.
  - Changed Scythe of Eternal Oblivion's projectile from Dragon Fireballs to Wither Skulls.
  - Changed the recipe for Substitute Jizo to use colored wool and leaves instead of hex colors.
  - `irons_spellbooks:citadel_keeper` can no longer be spawned from Apotheosis spawners. Utilize Hostile Neural Networks instead.
  - AE2 Annihilation Cores now use Null Matter instead of Electron Tube Singularities. Recipe now also outputs 2x Annihilation Cores instead of 1x. Also no longer require Mechanical Crafters to create.
  - Iron Golems no longer drop any Iron Ingots. Instead, they drop exactly 324x Iron Nuggets with and without Looting.
  - Final Scythe can now shoot both Wither Skulls and Ender Dragon Fireballs depending on if you are sneaking or not.
  - Eversor Gems can no longer be recycled into Unstable Entropy Particles.
  - Matchboxes now inherently have unlimited durability.
- 2026.02.15
  - Updated Neruina to `3.2.1` to potentially fix world crashing on save. **Please** let me know if there are issues.
- 2026.02.11
  - Made the following changes for Lunar:
    - It was my fault for not reading what the Eclipse Moon actually does, so this addresses the frustration with Eclipse Moon events entirely. I had assumed Eclipse Moon events would cover the world in a true dark sense and not give players Warden blindness. My deepest apologies. **Hopefully disabled the Eclipse Moon event by setting its weight from 50 to 0. If an Eclipse Moon does happen again, please let me know.**
    - You can no longer sleep during White Moon.
