It is now 2026.02.02. It's been a while.

While this is technically still a minor update, there are breaking changes.

### Fixed

- 2026.02.02
  - Fixed recipe conflict for compacting Crying Obsidian in a superheated Create Basin by changing its recipe entirely. See changes in the Changed section.

### Added

- 2026.02.02
  - Reinstated JEI.
  - Added some of SuperMartijn642's Fusion resource packs.
    - Fusion Block Transitions
    - Fusion Connected Glass
    - Fusion Emissive Ores

### Changed

- 2026.02.04
  - Enabled ModernFix's Dynamic Resources mixin to reduce RAM usage as this update has added memory overhead. Hopefully there are no future problems from this.
  - You can no longer sleep during Blood Moon, Hero Moon and Eclipse moon events from Lunar.
  - JEI now uses the correct button highlight color for Omnia.
- 2026.02.02
  - Updated Forge from 47.3.39 to 47.4.16
  - C.U.E superheated recipe now utilizes 1x Crying Obsidian, 1x Twisting Vines and 1x Weeping Vines.
  - AE2's Matter Cannon has been reverted back to its original recipe instead of requiring Mechanical Crafters with the only change being that it uses Iridium Ingots instead of Iron Ingots.
